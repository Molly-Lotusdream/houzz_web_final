### 项目描述
##### 本项目为HZ互联网家装平台，旨在为用户提升家装效率，用户可以在该平台浏览家装创意，联系平台上的设计师，购买家装产品，浏览博客，项目平台为Pc端，界面大都采用flex布局模型，并用Normalize.css进行了样式初始化

### 页面描述

index.html 项目主界面

login.html登录界面

blog.html网页博客界面

Browse Pros.html 用户浏览并获取设计师联系方式

ideas.html 用户获取家装创意界面

shop.html用户购买家具界面
